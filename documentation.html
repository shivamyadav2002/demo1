Quick Start

Get up and running with the DevHub API in just a few minutes. Here's a simple example to fetch popular assets:

cURL

curl -H "Authorization: Bearer YOUR_API_KEY" \
  -H "Content-Type: application/json" \
  https://api.devhub.com/v2/assets?category=components&limit=10

JavaScript

const response = await fetch('https://api.devhub.com/v2/assets', {
  headers: {
    'Authorization': 'Bearer YOUR_API_KEY',
    'Content-Type': 'application/json'
  }
});

const assets = await response.json();
console.log(assets);

JavaScript SDK

The official DevHub JavaScript SDK provides a convenient way to interact with our API from Node.js and browser environments.

Installation

npm

npm install @devhub/sdk

yarn

yarn add @devhub/sdk

Usage Example

import { DevHub } from '@devhub/sdk';

// Initialize client
const devhub = new DevHub({
  apiKey: 'dh_live_1234567890abcdef',
  environment: 'production' // or 'sandbox'
});

// Fetch assets
const assets = await devhub.assets.list({
  category: 'components',
  technology: 'react',
  limit: 10
});

// Get specific asset
const asset = await devhub.assets.get('ast_1234567890');

// Download asset
const downloadUrl = await devhub.assets.getDownloadUrl('ast_1234567890');

// Search
const searchResults = await devhub.search({
  query: 'login form',
  filters: {
    category: 'components'
  }
});

console.log('Assets:', assets);
console.log('Search Results:', searchResults);

Code Examples

Real-world examples showing how to integrate DevHub API into popular frameworks and platforms.

Next.js Integration

// pages/api/assets.js
import { DevHub } from '@devhub/sdk';

const devhub = new DevHub({
  apiKey: process.env.DEVHUB_API_KEY
});

export default async function handler(req, res) {
  try {
    const assets = await devhub.assets.list({
      category: req.query.category,
      limit: 20
    });
    
    res.status(200).json(assets);
  } catch (error) {
    res.status(500).json({ error: 'Failed to fetch assets' });
  }
}

  }
});

app.listen(3000);

Vue.js Integration

// composables/useDevHub.js
import { ref, onMounted } from 'vue';
import { DevHub } from '@devhub/sdk';

export function useDevHub() {
  const assets = ref([]);
  const loading = ref(false);
  
  const devhub = new DevHub({
    apiKey: 'your-api-key'
  });
  
  const fetchAssets = async (params = {}) => {
    loading.value = true;
    try {
      const data = await devhub.assets.list(params);
      assets.value = data;
    } catch (error) {
      console.error('Failed to fetch assets:', error);
    } finally {
      loading.value = false;
    }
  };
  
  return { assets, loading, fetchAssets };
}

Python Integration

import requests
from typing import List, Dict

class DevHubAPI:
    def __init__(self, api_key: str):
        self.api_key = api_key
        self.base_url = "https://api.devhub.com/v2"
        self.headers = {
            "Authorization": f"Bearer {api_key}",
            "Content-Type": "application/json"
        }
    
    def get_assets(self, **params) -> List[Dict]:
        response = requests.get(
            f"{self.base_url}/assets",
            headers=self.headers,
            params=params
        )
        response.raise_for_status()
        return response.json()["data"]

# Usage
client = DevHubAPI("dh_live_1234567890abcdef")
assets = client.get_assets(category="components", limit=10)

Best Practices

Error Handling

Always implement proper error handling to gracefully manage API failures and rate limits. 

async function fetchAssetsWithRetry(params, maxRetries = 3) {
  for (let attempt = 0; attempt < maxRetries; attempt++) {
    try {
      const response = await fetch('https://api.devhub.com/v2/assets', {
        headers: {
          'Authorization': 'Bearer ' + apiKey,
          'Content-Type': 'application/json'
        }
      });

      if (response.status === 429) {
        // Rate limited, wait and retry
        const retryAfter = response.headers.get('Retry-After') || 60;
        await new Promise(resolve => setTimeout(resolve, retryAfter * 1000));
        continue;
      }

      if (!response.ok) {
        throw new Error(`API Error: ${response.status} ${response.statusText}`);
      }

      return await response.json();
    } catch (error) {
      if (attempt === maxRetries - 1) {
        throw error;
      }
      // Wait before retry with exponential backoff
      await new Promise(resolve => setTimeout(resolve, Math.pow(2, attempt) * 1000));
    }
  }
}

Caching Strategy

Implement caching to reduce API calls and improve performance for your users. 

class DevHubCache {
  constructor(ttl = 300000) { // 5 minutes default TTL
    this.cache = new Map();
    this.ttl = ttl;
  }

  get(key) {
    const item = this.cache.get(key);
    if (!item) return null;
    
    if (Date.now() > item.expiry) {
      this.cache.delete(key);
      return null;
    }
    
    return item.data;
  }

  set(key, data) {
    this.cache.set(key, {
      data,
      expiry: Date.now() + this.ttl
    });
  }

  async fetchWithCache(url, options = {}) {
    const cacheKey = url + JSON.stringify(options);
    const cached = this.get(cacheKey);
    
    if (cached) {
      return cached;
    }
    
    const response = await fetch(url, options);
    const data = await response.json();
    
    this.set(cacheKey, data);
    return data;
  }
}

Troubleshooting

Common Error Codes

401 Unauthorized Invalid API Key

Check that your API key is correct and has the necessary permissions. Make sure you're including the "Bearer " prefix.

429 Too Many Requests Rate Limit Exceeded

You've exceeded your rate limit. Check the Retry-After header and implement exponential backoff.

404 Not Found Resource Not Found

The requested asset or resource doesn't exist or has been removed.

Debugging Tips

Enable Request Logging

Log all API requests and responses during development to identify issues quickly.

Check Response Headers

Important headers like X-RateLimit-Remaining and Retry-After provide useful debugging information.

Use Sandbox Environment

Test your integration against our sandbox API before going live.
